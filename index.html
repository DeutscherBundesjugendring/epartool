Redirection error

If you can read this then the redirection on the webserver is not working properly. All requests are to be redirected to the www folder, but this is apparently not happening.

Presently only Apache server is supported, if you are using something else, then it should be possible to set it up, but you are on your own.

If on Apache, please try:

  • Ensure the file .htaccess is present in the root of the project, that is in the same folder where this file is.
  • Ensure the file .htaccess has permissions set to be readable by the webserver.
  • Ensure .htaccess file processing is enabled in Apache configuration.
  • If nothing else works, please create an issue on GitHub.
